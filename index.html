What is the difference between Wallet and Wallet CLI?
First of all, both of Wallet and Wallet CLI allow you to send, receive and store Pitcoin digital currency. However, the wallet is a graphical program with a user interface and has no miner built-in. Wallet CLI is a command-line interface program without a graphical interface, meaning it runs in a terminal or command window, but has a miner built-in. The downloaded file is an archive consisting of several files, the most important of which is the daemon (Pitcoind) that runs the blockchain node and also allows mining of Pitcoin.
